# 智界雷达小程序产品优化方案

## 一、现状分析

### 当前功能模块
1. **首页** - 资讯列表（按时间排序，分类筛选）
2. **详情页** - 资讯内容 + 八维通洞察（AI生成）
3. **分析页** - 静态数据展示（热门话题、分类分布、趋势）
4. **我的** - 用户设置（未实际对接）

### 核心问题
1. **价值感知不足** - 用户不清楚"八维通洞察"的价值，点击率可能低
2. **数据孤岛** - 分析页数据为静态假数据，未与实际资讯关联
3. **粘性不足** - 缺少用户留存机制（收藏、历史、推送）
4. **商业闭环缺失** - 只有资讯输入，没有转化输出

---

## 二、优化方案（分优先级）

### P0 - 核心体验优化（立即实施）

#### 1. 详情页默认展示八维通洞察
**问题**：用户需要手动点击"获取八维通洞察"，增加操作成本，降低价值感知

**方案**：
- 进入详情页时自动调用AI生成洞察
- 显示加载态，提升期待感
- 将"八维通洞察"放在更明显的位置（标题下方）

**效果**：
- 提升洞察功能的使用率
- 强化"智界雷达 = 八维通专属AI顾问"的品牌定位

#### 2. 分析页接入真实数据
**问题**：当前数据为假数据，用户无法获得真实洞察

**方案**：
- 热门话题：统计资讯标题/内容中的高频词
- 分类分布：统计用户浏览/收藏的分类占比
- 趋势图表：展示近7天资讯数量趋势
- 添加"刷新"按钮，实时更新

**效果**：
- 提供真实的数据洞察
- 增加页面停留时间

#### 3. 收藏功能实际落地
**问题**：当前收藏功能为UI占位，未实际存储

**方案**：
- 使用本地存储（localStorage）保存收藏列表
- "我的"页面展示已收藏的资讯
- 详情页显示收藏状态

**效果**：
- 提升用户粘性
- 形成个人知识库

#### 4. 分享功能
**问题**：用户无法将有价值的资讯分享给同事或朋友圈

**方案**：
- 详情页添加"分享"按钮
- 支持分享到微信好友和朋友圈
- 生成分享卡片（标题+摘要+封面图）
- 首页也支持分享小程序

**技术实现**：
- 使用 Taro 的 `useShareAppMessage` 实现分享给朋友
- 使用 `useShareTimeline` 实现分享到朋友圈
- 创建 `ShareButton` 组件封装原生 Button 的 `openType="share"`
- 生成分享封面图存放在 `src/assets/share-cover.png`

**效果**：
- 扩大产品传播
- 形成口碑效应

## 五、总结

核心优化思路：
1. **降低使用门槛** - 洞察自动生成，无需手动触发

## 六、已完成的优化项

### 2024-01-XX: AI洞察优化 - 基于八维通官网信息

#### 背景
原来的AI洞察提示词较为通用，未结合八维通的具体业务，导致分析内容不够贴合公司实际需求。

#### 优化内容
1. **获取八维通公司信息**
   - 从官网（https://www.bwton.com）获取核心业务信息
   - 公司定位：卓越的空间智能服务商
   - 核心技术：城市数据资产底座、国产化参数几何建模内核、模拟仿真推演、数字孪生
   - 业务领域：智慧轨交（40+城市）、空间智能（100+城市）
   - 业务规模：300+业务场景、3000+数字化项目、150000+累计建模

2. **优化AI分析提示词**
   - 在 `getAnalysisPrompt` 方法中添加八维通公司背景信息
   - 要求AI分析时结合空间智能、数字孪生、智慧轨交等核心业务
   - 明确要求建议具体可执行

3. **优化默认分析结果**
   - 更新 `getDefaultAnalysis` 方法的默认返回内容
   - 使默认内容更贴合八维通业务特点

4. **优化内容生成提示词**
   - 详情页内容生成时关注空间智能、数字孪生、智慧城市相关内容

#### 代码修改
- 文件：`server/src/modules/news/news.controller.ts`
- 修改方法：`getAnalysisPrompt`、`getDefaultAnalysis`
- 修改提示词：详情页内容生成

#### 效果预期
- AI分析更贴合八维通业务方向
- 洞察建议更具可操作性
- 提升用户对"八维通洞察"的价值感知

### 2024-01-XX: 实时资讯抓取优化

#### 背景
确保所有资讯都是实时抓取的全网最新最热门内容，提升资讯价值。

#### 优化内容
1. **多维度搜索策略**
   - 使用多个搜索关键词组合并行搜索
   - 热点新闻：具身智能、人形机器人、空间智能等热点词
   - 技术突破：技术突破、最新发布、研发成果
   - 政策动态：政策法规、指导意见、最新通知
   - 市场趋势：投资融资、市场规模、行业报告

2. **结果合并与去重**
   - 并行执行多个搜索请求
   - 按标题去重，确保资讯不重复
   - 按发布时间降序排列，最新的排在前面

3. **质量过滤**
   - 过滤科普类、百科类、问答类内容
   - 过滤标题过短的低质量资讯
   - 过滤低质量来源（百科、问答平台）

4. **详情页增强**
   - 先搜索获取更多实时信息
   - 结合搜索结果生成更准确的详细内容

#### 代码修改
- 修改文件：`server/src/modules/news/news.controller.ts`
- 新增方法：`getSearchQueries()`、`isLowQuality()`
- 优化逻辑：并行搜索、结果合并、智能排序

#### 效果预期
- 所有资讯均为实时抓取的最新内容
- 按时间排序，确保最新的资讯排在前面
- 提升资讯的时效性和价值感
