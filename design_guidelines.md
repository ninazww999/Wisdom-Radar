# 具身智能资讯助手 - 设计指南

## 品牌定位

**应用定位**：面向董事长/决策层的具身智能与空间智能领域专业资讯智能助手
**设计风格**：现代简约、深色主题、高级质感、科技感
**目标用户**：企业董事长、高管、决策层

## 设计理念

融合 **Linear** 的极简设计与 **抖音** 的深色沉浸体验：
- 深色主题减少视觉疲劳，适合长时间阅读
- 微妙渐变增添科技感和高级感
- 大圆角卡片营造现代感
- 精致的分割线和层次感
- 流畅的动画过渡

## 间距系统

使用 Tailwind 间距单位（1单位 = 4px）：

- **页面边距**：`px-5`（20px）
- **卡片内边距**：`p-5`（20px）
- **组件间距**：`gap-4`（16px）
- **列表项间距**：`mb-4`（16px）

## 组件使用原则

### 组件选型约束

**必须优先使用** `@/components/ui/*` 中的通用组件：

- **按钮**：`Button`（主次操作、快捷入口）
- **输入框**：`Input`（搜索框、反馈输入）
- **卡片**：`Card`、`CardContent`（资讯卡片、数据卡片）
- **标签**：`Badge`（资讯分类标签、状态标签）
- **标签页**：`Tabs`（资讯分类切换、时间维度切换）
- **弹窗**：`Dialog`（确认操作、详情预览）
- **提示**：`Toast`（操作反馈、状态提示）
- **加载态**：`Skeleton`（资讯列表加载、数据加载）
- **分割线**：`Separator`（内容分割）

### 高级视觉效果

- **渐变背景**：头部、按钮使用紫色渐变
- **玻璃态效果**：`backdrop-blur` 模糊背景
- **圆角**：`rounded-2xl`（16px）大圆角
- **阴影**：深色主题使用内阴影或发光效果
- **边框**：`border border-gray-800` 微妙边框

## 导航结构

### TabBar 配置

| 页面   | 路径                | 图标       | 说明                     |
| ------ | ------------------- | ---------- | ------------------------ |
| 首页   | `pages/index/index` | `House`    | 今日推送、资讯分类入口   |
| 分析   | `pages/analysis/index` | `TrendingUp` | 行业趋势、数据可视化     |
| 我的   | `pages/profile/index` | `User`     | 订阅管理、历史记录、设置 |

### 深色TabBar样式

- 背景色：`gray-950`（#030712）
- 未选中图标：`gray-500`（#6b7280）
- 选中图标：`violet-500`（#8b5cf6）
- 文字：跟随图标颜色

## 容器样式原则

- **卡片圆角**：`rounded-2xl`（16px）
- **卡片边框**：`border border-gray-800`
- **卡片背景**：`bg-gray-900`
- **悬停效果**：`hover:bg-gray-800`（交互反馈）
- **列表卡片间距**：`mb-4`（16px）

## 状态展示原则

### 空状态

- 使用 `Card` + 居中图标
- 图标颜色：`gray-600`
- 文字颜色：`gray-400`
- 提示文案简洁明确

### 加载态

- 使用 `Skeleton` 组件
- 背景色：`bg-gray-800`
- 动画：`animate-pulse`

### 错误状态

- 使用 `Toast` 或 `Alert` 提示
- 错误色：`rose-500`
- 明确错误原因

## 小程序约束

### 包体积优化

- 图片使用 CDN 或对象存储
- 避免大图本地存储
- 使用分包加载（如需）

### 性能优化

- 使用虚拟列表（长列表场景）
- 图片懒加载：`lazyLoad`
- 避免深层嵌套

### 微信小程序特有

- TabBar 图标必须使用本地 PNG 文件
- 深色主题需配置 `navigationBarBackgroundColor`
- 使用 CLI 工具生成图标

## 关键页面设计要点

### 首页

- 顶部：渐变背景 + 日期 + 标题
- 快捷入口：圆角图标卡片
- 资讯卡片：深色卡片 + 渐变标签 + 悬停效果
- AI助手：玻璃态卡片 + 渐变边框

### 资讯详情页

- 顶部：渐变返回栏
- 内容区：深色卡片 + 微妙边框
- AI解读：渐变背景 + 发光效果
- 操作栏：固定底部 + 玻璃态

### 智能分析页

- 顶部：渐变标题栏
- Tab切换：胶囊样式
- 数据卡片：渐变边框 + 悬停效果
- 图表区域：深色背景 + 发光线条

### 我的页面

- 顶部：渐变用户信息卡片
- 设置列表：深色卡片组
- 图标：彩色图标 + 深色背景
- 退出按钮：红色主题
