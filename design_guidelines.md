# 具身智能资讯助手 - 设计指南

## 品牌定位

**应用定位**：面向董事长/决策层的具身智能与空间智能领域专业资讯智能助手
**设计风格**：专业、简洁、高效、可信
**目标用户**：企业董事长、高管、决策层

## 间距系统

使用 Tailwind 间距单位（1单位 = 4px）：

- **页面边距**：`p-4`（16px）
- **卡片内边距**：`p-4`（16px）
- **组件间距**：`gap-4`（16px）或 `gap-3`（12px）
- **列表项间距**：`gap-3`（12px）

## 组件使用原则

### 组件选型约束

**必须优先使用** `@/components/ui/*` 中的通用组件：

- **按钮**：`Button`（主次操作、快捷入口）
- **输入框**：`Input`（搜索框、反馈输入）
- **卡片**：`Card`、`CardContent`、`CardHeader`、`CardTitle`（资讯卡片、数据卡片）
- **标签**：`Badge`（资讯分类标签、状态标签）
- **标签页**：`Tabs`（资讯分类切换、时间维度切换）
- **弹窗**：`Dialog`（确认操作、详情预览）
- **提示**：`Toast`（操作反馈、状态提示）
- **加载态**：`Skeleton`（资讯列表加载、数据加载）
- **分割线**：`Separator`（内容分割）
- **进度条**：`Progress`（阅读进度、数据进度）

**页面结构容器**使用 Taro 原生组件：

- `View`：页面容器、布局容器
- `Text`：文本展示
- `Image`：图片展示
- `ScrollView`：滚动容器

### 页面实现前检查清单

在编写任何页面 JSX 前，必须逐项检查：

- [ ] 页面需要的按钮、输入框、卡片、标签、Tabs、弹窗、Toast、加载态是否已映射到 `@/components/ui/*`
- [ ] 不存在用 `View/Text` 手搓按钮、输入框、卡片、标签、Tabs、弹窗等通用 UI
- [ ] 容器样式（圆角、阴影、内边距）遵循指南
- [ ] 间距使用 `gap-*` 而非硬编码 `px` 值

## 导航结构

### TabBar 配置

| 页面   | 路径                | 图标       | 说明                     |
| ------ | ------------------- | ---------- | ------------------------ |
| 首页   | `pages/index/index` | `House`    | 今日推送、资讯分类入口   |
| 分析   | `pages/analysis/index` | `BarChart3` | 行业趋势、数据可视化     |
| 我的   | `pages/profile/index` | `User`     | 订阅管理、历史记录、设置 |

### 页面跳转规范

- TabBar 页面跳转：使用 `Taro.switchTab()`
- 普通页面跳转：使用 `Taro.navigateTo()`
- 详情页路径示例：`/pages/detail/index?id=xxx`

## 容器样式原则

- **卡片圆角**：`rounded-lg`（8px）
- **卡片阴影**：`shadow-sm`
- **卡片边框**：`border border-gray-200`
- **列表卡片间距**：`mb-3`（12px）

## 状态展示原则

### 空状态

- 使用 `Card` + 居中文字
- 提示文案简洁明确
- 配合图标（如 `Inbox`）

### 加载态

- 使用 `Skeleton` 组件
- 资讯列表：卡片骨架屏
- 数据加载：进度条或骨架屏

### 错误状态

- 使用 `Alert` 或 `Toast` 提示
- 明确错误原因
- 提供重试操作

## 小程序约束

### 包体积优化

- 图片使用 CDN 或对象存储
- 避免大图本地存储
- 使用分包加载（如需）

### 性能优化

- 使用虚拟列表（长列表场景）
- 图片懒加载：`lazyLoad`
- 避免深层嵌套

### 微信小程序特有

- TabBar 图标必须使用本地 PNG 文件
- 使用 CLI 工具生成图标：`npx taro-lucide-tabbar`
- 图标路径必须以 `./` 开头

## 关键页面设计要点

### 首页

- 顶部：日期 + 快捷入口（政策、行业、技术、市场）
- 中部：今日推送卡片列表
- 底部：TabBar 导航
- 资讯卡片：标题 + 来源 + 时间 + 分类标签 + 摘要

### 资讯详情页

- 顶部：返回按钮 + 标题
- 中部：原文内容 + AI 解读
- 底部：相关推荐 + 操作栏（收藏、分享）

### 智能分析页

- 顶部：时间维度切换（本周/本月/本季度）
- 中部：趋势图表 + 关键数据卡片
- 底部：决策建议列表

### 我的页面

- 顶部：用户信息
- 中部：订阅管理、历史记录、设置入口
- 底部：退出登录
